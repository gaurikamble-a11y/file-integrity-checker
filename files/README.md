# 🛡️ File Integrity Checker

This is a cybersecurity project that monitors files and detects unauthorized changes using SHA-256 hashing.

## Features
- Detects modified files
- Detects deleted files
- Detects new files
- Real-time monitoring
- Logs system activity

## Tech Used
- FastAPI (Backend)
- React (Frontend)
- SHA-256 hashing

## How it works
The system creates a baseline of file hashes and continuously compares them to detect any changes.
